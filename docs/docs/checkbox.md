# Widget - CheckBox / 组件 - 单选框

标签: `<swift-checkbox>`

#### On

示例: 
```html
<swift-checkbox state="on">Label</swift-checkbox>
```

效果 (可点击): <swift-checkbox state="on">Label</swift-checkbox>

#### Disabled / 禁用

示例: 
```html
<swift-checkbox state="on" disabled="true">Label</swift-checkbox>
```

效果 (可点击): <swift-checkbox state="on" disabled="true">Label</swift-checkbox>

### Off

示例: 
```html
<swift-checkbox state="off">Label</swift-checkbox>
```

效果 (可点击): <swift-checkbox state="off">Label</swift-checkbox>

#### Disabled / 禁用

示例: 
```html
<swift-checkbox state="off" disabled="true">Label</swift-checkbox>
```

效果 (可点击): <swift-checkbox state="off" disabled="true">Label</swift-checkbox>
