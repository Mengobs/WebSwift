# Container - Messagebox / 框架 - 弹窗

标签: `<swift-messagebox>`

示例: 
```html
<swift-messagebox iconpath="../../images/访达.png">
    <p slot="title">Title</p>
    <p slot="text">Text</p>
    <swift-button primary slot="button">Button</swift-button>
</swift-messagebox>
```
