# Widget - Button / 组件 - 按钮

标签: `<swift-button>`

### Primary

示例: 
```html
<swift-button primary>Label</swift-button>
```

效果 (可点击): <swift-button primary>Label</swift-button>

#### Disabled / 禁用

示例: 
```html
<swift-button primary disabled="true">Label</swift-button>
```

效果 (可点击): <swift-button primary disabled="true">Label</swift-button>

### Normal

示例: 
```html
<swift-button>Label</swift-button>
```

效果 (可点击): <swift-button>Label</swift-button>

#### Disabled / 禁用

示例:
