# Widget - Switch / 组件 - 开关

标签: `<swift-switch>`

#### On

示例: 
```html
<swift-switch state="on"></swift-switch>
```

效果 (可点击): <swift-switch state="on"></swift-switch>

#### Disabled / 禁用

示例: 
```html
<swift-switch state="on" disabled="true"></swift-switch>
```

效果 (可点击): <swift-switch state="on" disabled="true"></swift-switch>

### Off

示例: 
```html
<swift-switch state="off"></swift-switch>
```

效果 (可点击): <swift-switch state="off"></swift-switch>

#### Disabled / 禁用

示例: 
```html
<swift-switch state="off" disabled="true"></swift-switch>
```

效果 (可点击): <swift-switch state="off" disabled="true"></swift-switch>
